Spring MVC with Ruby

Compile:

#+BEGIN_EXAMPLE
  $ mvn package
#+END_EXAMPLE

Run:

#+BEGIN_EXAMPLE
  $ mvn jetty:run
#+END_EXAMPLE

The app can then be accessed there: http://localhost:8080/myapp/welcome
